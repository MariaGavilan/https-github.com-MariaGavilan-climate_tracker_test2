## Project Overview

This repository contains a complete suite of MATLAB tools for:

1. **Carbon Footprint Calculation:**  Measure emissions across transport, home energy, food, digital, and consumption
2. **Regional Context Analysis:**  Compare your footprint against regional and global averages
3. **Action Planning:**  Explore 50+ evidence-based reduction strategies
4. **Interactive Visualization:**  Web-based dashboard for real-time impact assessment
5. **Paris Agreement Alignment:**  Track progress toward 2030 climate targets


### Prerequisites

- MATLAB R2024b or later (R2025a recommended for GitHub integration)
- MATLAB Online (preferred) or Desktop installation
